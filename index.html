How to use this template

  1. Open the HTML file and replace topfindsind05-21 with your Amazon Associates tag (example: yourtag-21).
  2. Replace the sample products array in the script with your products (ASIN, title, image, price, description).
  3. Upload the file to your hosting (Netlify/Hostinger/Vercel or GitHub Pages). Use HTTPS so Pinterest can fetch images.
  4. Create Pins: Use the provided "Pin" button on each card or manually pin product images to Pinterest with page URL and image URL.
